# Customer Churn Analysis Dashboard (Power BI)
 Project Overview 

This project is a Power BI Dashboard built to analyze customer churn behavior and identify key factors influencing customer retention and attrition.

It helps businesses understand why customers leave, which customer segments are at risk, and what factors contribute to churn.

Objective

The main goal of this project is to:

Analyze customer churn patterns
Identify high-risk customer segments
Understand impact of subscription type, contract length, and demographics
Provide actionable business insights for customer retention

 Dataset

Total Records: ~64,000 customers
Key Fields:
Customer ID
Age
Gender
Subscription Type
Contract Length
Tenure
Support Calls
Payment Delay
Churn Status

Tools Used

Power BI
Power Query (Data Cleaning)
DAX (Calculated Measures)
Data Visualization

Key KPIs

Total Customers
Churned Customers
Retained Customers
Churn Rate
Average Support Calls
Average Payment Delay

Visualizations Used

Bar Charts (Churn by Subscription, Contract Type, Age Group)
Gauge Chart (Overall Churn Rate)
Line Chart (Trend Analysis)
KPI Cards
Slicers for Interactivity

Key Insights

Basic subscription users have the highest churn rate
Monthly contracts show more churn compared to quarterly and annual plans
Customers with higher payment delays are more likely to churn
Adult customers show higher churn behavior
Loyal customers also contribute significantly to churn, indicating hidden dissatisfaction

Interactive Filters (Slicers)

Gender
Age Group
Subscription Type
Contract Length
Tenure Group

These slicers allow dynamic exploration of customer segments.

Business Impact

This dashboard helps organizations to:

Improve customer retention strategies
Identify high-risk customer segments
Optimize subscription plans
Reduce churn rate through data-driven decisions
